Open Site Review

Compatibility and front-end practice scanner for HTML, CSS, and JavaScript.

What it does

  • Checks whether you are using features that are too new for your target browsers.
  • Identifies polyfills that are no longer needed.
  • Highlights native web platform features that can replace JavaScript.

How to request a scan

Open a GitHub issue in the open-site-review repository with a title beginning with SCAN: and include the URLs you want reviewed in the issue body. The workflow will run automatically, post the report as a comment, and publish it here.
